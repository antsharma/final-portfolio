### Setup

Get the code

<pre>git clone https://github.com/ubaimutl/react-portfolio.git</pre>
 
Install required dependencies

<pre>npm install</pre>
<pre>or</pre>
<pre>yarn install</pre>


Start the server
<pre>npm start</pre>
<pre>or</pre>
<pre>yarn start</pre>
